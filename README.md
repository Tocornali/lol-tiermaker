# 🏆 LoL TierMaker

A professional, high-performance desktop application for creating and managing League of Legends Tier Lists. Built with a focus on **Premium UI/UX**, fluid animations, and robust multi-project management.

![React](https://img.shields.io/badge/React-v19-61dafb)
![Tailwind](https://img.shields.io/badge/Tailwind-v4-38b2ac)

## ✨ Key Features

- **📂 Multi-Project Architecture**: Create, rename, and manage multiple Tier Lists simultaneously using a dynamic tabbed interface.
- **🎨 Signature Aesthetics**: Exclusive **Graphite & Turquoise** theme with a frameless, rounded-corner window design.
- **⚡ Real-time Riot Integration**: Automatically fetches the latest champion data and high-quality icons from Riot Games' Data Dragon API.
- **🎯 Precision Drag & Drop**: Advanced placement logic with visual indicators for pixel-perfect champion ordering.
- **💾 Persistent State**: Your projects and rankings are automatically saved locally, ensuring no progress is ever lost.
- **🎬 Fluid Animations**: Powered by `framer-motion` for organic tab transitions and interactive UI elements.

## 🛠️ Technology Stack

- **Core**: [Electron](https://www.electronjs.org/) & [Vite](https://vitejs.dev/)
- **Frontend**: [React](https://reactjs.org/) & [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Custom Design Tokens)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **API**: Riot Games Data Dragon

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Tocornali/lol-tiermaker.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the application in development mode:
```bash
npm run dev
```

### Production Build

Generate a production-ready installer for your OS:

```bash
# Windows
npm run build:win

# macOS
npm run build:mac

# Linux
npm run build:linux
```
